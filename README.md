# nimingban

## 简介

准备使用`SvelteKit`搭建一个类似于A岛（已停止运营）或者2chan，4chan的匿名版论坛

后端数据库使用`PostgresSQL`

## 资源设计

| 功能  | 路径（和请求方法）        | 携带参数 |
| ----  | ---------------         | ------- |
| 主页  | GET /                    |         |
| 某个版 | GET /board/{board_name} |         | 

## 参数设置

### 版面参数

* 版规
* 是否允许发帖
* 允许组/禁止组
* 发帖间隔
* 串最大长度
* 串附件图片最大数
* 串图片大小限制

### 用户参数

* 分组 
    * ADMIN 管理员
    * USER 普通用户
    * GUEST 分配账户的普通用户
    * NOLOGIN 被禁止登录的用户
